Position

Static

  • positioned to the normal flow of document
  • no offset and layering works
  • space given does not change
  • no affect on other element

Relative

  • positioned to the normal flow of document
  • top, right, bottom, left, z-index offset works
  • space given does not change
  • no affect on other element

Absolute

  • fixed on ancestor
  • remove from the normal flow of document
  • top, right, bottom, left, z-index offset works
  • no space is given
  • positioned relative to closest position ancestor
ABSOLUTE

Fixed

  • fixed on viewport
  • remove from the normal flow of document
  • top, right, bottom, left, z-index offset works
  • no space is given
  • positioned relative to closest position ancestor
FIXED

Sticky

  • static but will then be fixed after scroll
